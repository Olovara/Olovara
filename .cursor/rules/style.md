Project uses shadcn/ui components.

Follow OLOVARA design principles:
- Minimal Swiss design
- Lots of white space
- Do not add excessive UI clutter

Rules:

Reference STYLE_GUIDE markdown file in docs folder.

Always use shadcn components when possible.

Do not create custom button styles.
Use the shadcn Button component.

Use brand colors defined in Tailwind.
Never hardcode hex colors.

Typography rules:
- font-sans (Jost) for UI
- font-serif (Noto Serif Display) only for some hero text.

Layouts must prioritize whitespace and simplicity.
Avoid dense ecommerce layouts.
